## Install

- Install Node.js on your computer, it should have NPM.js with it
- Install git on your computer
- Install VSCode and git should be in it, you can manually push your commits or you can use a SSH that takes care of it for you just with some clicks
- Go to your desired folder in your file explorer 
- Clone the repository with `git clone https://github.com/devonyoung-blip/CMPE131-EVA.git` (HTTP) or `git clone git@github.com:devonyoung-blip/CMPE131-EVA.git` (SSH) and go intot the newly created forlder and open it in VSCode
- Right-click on the folder name and click `Open in Integrated Terminal`
- A window opens at the bottom with `[foldername] git:(main)`, go there
- Type `npm install` or `sudo npm install` (protected root) and enter
